# Equipment CRUD App

Full-stack CRUD application built with React and Node.js.

## Features
- Create, update, delete records
- REST API backend
- Frontend with dynamic forms

## Tech Stack
- React
- Node.js (Express)
- MongoDB (or in-memory)

## Purpose
This project demonstrates my ability to build and structure full-stack applications, based on my internship experience working with CRUD systems.
