# Equipment CRUD App

Full-stack CRUD application built with React and Node.js.

## Features
- Create, view, and delete records
- REST API using Node.js and Express
- Simple frontend built with React

## Tech Stack
- React
- Node.js (Express)

## Description
This project simulates a real-world record management system. It demonstrates how a frontend interacts with backend APIs to perform CRUD operations.

## Note
This project reflects my internship experience working on CRUD-based systems. The original code is not publicly shareable, so I recreated similar functionality here.
